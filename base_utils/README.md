# Base Utilities

Questa cartella contiene utilità generali suddivise per funzionalità.

## Sottocartelle

- `cifs/` - script per testare condivisioni di rete SMB/CIFS e gestire le credenziali.
- `network/` - script per test di rete, ping e risoluzione DNS.
- `misc/` - script generici e prototipi di interfaccia per funzioni bash.

## Come usare

Entra nella sottocartella della funzionalità che ti serve e leggi il `README.md` locale.

## Esempio

```bash
cd base_utils/cifs/bash
bash check_cifs.sh
```
