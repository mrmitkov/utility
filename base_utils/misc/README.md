# Misc Utilities

Questa cartella contiene script generici ed esempi di funzioni bash interattive.

## Script disponibili

- `function.sh` - esempio di menu testuale per selezionare ed eseguire funzioni bash.
- `function2_test.sh` - esempio che utilizza `dialog` per selezioni a casella di controllo.

## Uso

Per `function.sh`:

```bash
cd base_utils/misc
bash function.sh
```

Per `function2_test.sh`:

```bash
cd base_utils/misc
bash function2_test.sh
```

## Note

- `function2_test.sh` richiede `dialog` installato per il menu grafico in terminale.
- Questi script sono utili come base per sviluppare altre interfacce di selezione e automazione.
